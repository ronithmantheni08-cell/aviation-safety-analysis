# Aviation Safety: FAA Wildlife Strike Analysis
**Technical Stack:** Python (Pandas, Matplotlib), Jupyter Notebook, Big Data Analytics

## 📌 Project Overview
This analysis explores over 100,000 FAA records to identify patterns in wildlife-aircraft collisions. The goal was to pinpoint high-risk airports and flight phases to support better safety protocols.

## ⚙️ The Technical Process
- **Data Engineering:** Handled "Big Data" challenges, including encoding fixes (Latin-1) and large-scale null imputation for aircraft mass and flight height.
- **Statistical Analysis:** Aggregated data by State and Airport ID to rank the most frequent incident locations.
- **Visualization:** Generated trend charts using Matplotlib to visualize the correlation between "Phase of Flight" and strike severity.

## 💡 Key Findings
- Identified the **Top 10 Airports** by incident frequency, providing a focused list for resource allocation.
- Discovered that the **Approach and Landing phases** account for the majority of strikes, suggesting a need for increased bird-deterrent measures near runways.
