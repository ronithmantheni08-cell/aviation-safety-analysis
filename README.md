# Aviation Safety: FAA Wildlife Strike Analysis
**Technical Stack:** Python (Pandas, Matplotlib), Jupyter Notebook, Big Data Analytics

## 📌 Project Overview
This analysis explores a massive dataset of over 100,000 FAA records to identify patterns in wildlife-aircraft collisions. I used data cleaning and statistical modeling to pinpoint high-risk airports and flight phases, providing a data-backed foundation for airport safety protocols.

## ⚙️ The Technical Process
- **Data Engineering:** Resolved "Big Data" challenges, including encoding fixes (Latin-1) and large-scale null imputation for aircraft mass and flight height.
- **Statistical Analysis:** Aggregated data by State and Airport ID to rank the most frequent incident locations.
- **Visualization:** Generated trend charts using Matplotlib to visualize the correlation between "Phase of Flight" and strike severity.

## 💡 Key Findings
- **High-Risk Zones:** Identified the Top 10 Airports by incident frequency, allowing for targeted resource allocation.
- **Critical Flight Phases:** Discovered that the **Approach and Landing phases** account for the majority of strikes, suggesting a need for increased bird-deterrent measures near runways.
